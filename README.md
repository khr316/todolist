# TodoList

## 프로젝트 기능

- **할 일 추가**: 사용자가 새로운 할 일을 입력하여 추가할 수 있습니다
- **할 일 수정**: 추가된 할 일의 내용을 수정할 수 있습니다
- **할 일 삭제**: 완료되거나 필요 없는 할 일을 삭제할 수 있습니다
- **할 일 상태 변경**: 할 일을 완료 상태로 변경하거나, 완료된 할 일을 다시 미완료 상태로 변경할 수 있습니다

## 사용 기술

- **프론트엔드**: css, html, javascript
- **백엔드**: spring boot
- **데이터베이스**: MySQL
- **버전 관리**: Git
